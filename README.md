# IndexGT
- *IndexGT : le site web avec du FastHTML et du Tour de France dedans*, 2025, *Programmez!* n°267, pp. 44-47.
- D’un point de vue fonctionnel, cette petite application web établit un classement par pays en fonction du nombre de coureurs encore en activité ayant remporté au moins une étape sur le Tour de France, le Giro ou la Vuelta.
